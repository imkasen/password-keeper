# Password Keeper

* 安卓选修课期末作业
* 功能是保存账户密码，类似于 Ipassword
* 只做了基本界面和简单的保存密码功能

## 欠缺功能

* 账户分类
* 账户信息的逐项修改与删除
* 等等
